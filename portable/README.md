# 智狼 (MindWolf) - 便携版

这是智狼 (MindWolf) AI狼人杀游戏的便携版本。

## 使用方法

1. 双击 mindwolf.exe 启动应用
2. 首次运行会在当前目录下创建 config 目录存储配置
3. 如需重置配置，删除 config 目录即可

## 系统要求

- Windows 10 或更高版本
- x64 架构

## 文件说明

- mindwolf.exe: 主程序文件 (约 5.6 MB)
- config/: 配置文件目录 (首次运行后自动创建)
- logs/: 日志文件目录 (首次运行后自动创建)
